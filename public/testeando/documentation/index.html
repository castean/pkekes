Detail.html

- The rating star

Declare HTML:

<span class="rating-static rating-30"></span>

It will show the rating bar with 3 stars rating. See the table below to change the rating star:

rating-50 5 stars                                                    
rating-40 4 stars
rating-30 3 stars
rating-20 2 stars
rating-10 1 star
rating-0 0 star
rating-5 0.5 star
rating-15 1.5 stars
rating-25 2.5 stars
rating-35 3.5 stars
rating-45 4.5 stars

-  The rating bar

Declare HTML:

<span class="bar bar5" style="WIDTH">&nbsp;</span>

bar5: the number of ratings with 5 stars

bar4: the number of ratings with 4 stars

bar3: the number of ratings with 3 stars

bar2: the number of ratings with 2 stars

bar1: the number of ratings with 1 star

 

Formula to calculate with of rating bar:

WIDTH = ((Number of rates * 100)/Total rates) * 2;

Ex: WIDTH = ((1,967 * 100) / 2908) * 2 = 135.28px;

 

Category.html

- Refine your seach

You can find the Refind search box on the left of category.html. I had redesigned the bootstrap Nav lists.

There were two ways to create Refine search in this theme: multi choice and one choice.

You add the class refine-search-onechoice for once choice, and the class refine-search-multichoice for multi choice

EX:<ul class="nav nav-list refine-search-onechoice">...</ul>

 

Pricing.html

I created 4 color styles for this theme: Orange, Green, Blue, Red.

You can switch between these color by change the class name.

Example:
<div class="orange">....</div> => use Orange style
To switch Red style:
<div class="red">....</div>
